# HW-TMS-NativeJS(Task-7)
Task1 Найти первое число, которое нарушает последовательность.
Создать массив, и вернуть первое число, которое выбивается из последовательности.
Если такого числа нет - возвращаем null.
Примеры для массивов:
[1, 2, 3, 4, 6, 7, 8] => 6
[4, 5, 6, 7, 8, 10, 11, 14] => 10
[2, 3, 4] => null
Добавить логику, чтобы при двух и более нарушений возвращался массив цифр.

Task2 Аббривиация
Пишем функцию, которая принимает любое количество строк и
Возвращает аббревиатуру из слов. Если в переданной строке есть пробелы, каждое слово в строке тоже должно возвращать аббревиатуру.
Примеры
makeAbbr(‘Sam’, ‘Houser’) => ’S.H.’
makeAbbr(’Teach Me Skills, ‘The best IT school’) => ’T.M.S.T.B.I.S’

Task3 Проверка скобок на закрывающие.
Пишем функцию, в которую приходит строка, с произвольным количеством открывающих и закрывающих скобок. 
На выходе функция должна возвращать true - если все скобки закрыты, или false - если нет.
Пример ‘(2 + 2) * 2’ => true   	‘7 + (76 * (23 - 9)’ => false

Task4 Выбрать из массива числа без повторений.
Пример 
[1, 1, 2, 3, 4, 6, 8, 8, 9, 14, 15] => [2, 3, 4, 6, 9, 14, 15];
Добавить к задаче ещё условие, чтобы возвращались не только уникальные числа, но и простые (1, 2, 3, 5, 7, 11, 13, 17, 19 и тд).
let mass = [1, 1, 2, 3, 4, 6, 8, 8, 9, 14, 15];
